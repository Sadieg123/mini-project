# Daily Habit Tracker

A lightweight habit tracking web application built using HTML, CSS, and JavaScript. This project focuses on client-side state management using localStorage, allowing users to persist data without a backend.

---

## Live Demo
https://sadieg123.github.io/Portfolio-Reflection/

---

## Features

- Add new daily habits
- Mark habits as complete
- Delete habits
- Persistent storage using localStorage
- Responsive design for mobile and desktop
- Clean, minimal UI with visual feedback for completed tasks

---

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox)
- JavaScript (Vanilla JS)
- localStorage API

---
